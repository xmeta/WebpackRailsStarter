=== Requirements
////

////

* Ruby 2.3.1
* Rails 5
* PostgreSQL

=== Supported Platforms
* Linux
* Windows
* MacOS

=== Installation
// インストール方法を書く．
----
git clone https://github.com/xmeta/WebpackRailsStarter.git
cd WebpackRailsStarter
bundle install
npm install
----

=== Usage
----
foreman start
